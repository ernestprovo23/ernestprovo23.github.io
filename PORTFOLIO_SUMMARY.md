## 🚀 Quick Deployment Guide

### Option 1: Create a New Repository (Recommended)
1. Create a new repository on GitHub called `portfolio` or `my-portfolio`
2. Push these files to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio setup"
   git branch -M main
   git remote add origin https://github.com/ernestprovo23/portfolio.git
   git push -u origin main
   ```

3. Go to repository Settings → Pages
4. Select "Deploy from a branch" → "main" → "/ (root)"
5. Your portfolio will be live at `https://ernestprovo23.github.io/portfolio`

### Option 2: Use Username Repository
1. Create a repository named `ernestprovo23.github.io`
2. Push these files to the main branch
3. Your portfolio will be live at `https://ernestprovo23.github.io`
